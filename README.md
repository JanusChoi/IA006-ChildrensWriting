本文为开智学堂信息分析六期结业作业说明文档。本次要分析的课题为：少儿写作。

## 开题综述

结合阳老师建议，初定分析主题为：少儿写作培训机构的全局认识

- 商业：市场上现有的产品分析，校内外写作体系

- 少儿写作需针对的哪个年龄段的孩子？

- 在写作方面学生面临的主要问题是什么？该如何解决？
